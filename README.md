# Build a simple full stack todos app 
- Front end: React
- Back end: ExpressJS, NodeJS
- Database: MySQL

## Configuration
Please create a `.env` file before starting server.

`.env` example
```
HOST="localhost"
DB_USER="your_db_user"
PASSWORD="your_db_password"
```

## Start server
```
npm run server
```

## Start front end web
```
npm start
```
